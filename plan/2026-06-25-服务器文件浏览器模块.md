## 步骤 1：扩展 FTP/SFTP 客户端 — 新增 delete 和 rename

**文件**: `src/main/services/ftp-client.ts`

TransferClient 接口新增：
```ts
delete(remotePath: string): Promise<void>
rename(oldPath: string, newPath: string): Promise<void>
```

FTPClient 实现：
- `delete`: `this.client.remove(remotePath)` (basic-ftp 内置)
- `rename`: `this.client.rename(oldPath, newPath)` (basic-ftp 内置)

SFTPClient 实现：
- `delete`: `this.client.delete(remotePath)` 删除文件 + `this.client.rmdir(remotePath)` 删除目录
- `rename`: `this.client.rename(oldPath, newPath)`

## 步骤 2：新增 IPC Handler

**文件**: `src/main/ipc/index.ts`

新增 `registerBrowserIPC(configStore)` 函数，每个 handler 创建临时客户端 → 连接 → 操作 → 断开：

| IPC Channel | 参数 | 返回值 | 说明 |
|-------------|------|--------|------|
| `browser:list` | `{ serverId, remotePath }` | `RemoteFileInfo[]` | 列出目录内容 |
| `browser:mkdir` | `{ serverId, remotePath, dirName }` | `{ success }` | 新建文件夹 |
| `browser:delete` | `{ serverId, remotePath }` | `{ success }` | 删除文件/文件夹 |
| `browser:rename` | `{ serverId, oldPath, newPath }` | `{ success }` | 重命名 |
| `browser:download` | `{ serverId, remotePath, localPath }` | `{ success }` | 下载文件（通过 dialog 选保存路径） |

辅助函数 `getClient(configStore, serverId)`: 根据 serverId 获取 ServerConfig → createClient → connect → 返回 { client, config }。

**文件**: `src/main/index.ts` — `registerIPC()` 中增加 `registerBrowserIPC(this.configStore)` 调用。

## 步骤 4：新建 ServerBrowser 页面

**文件**: `src/renderer/pages/ServerBrowser.tsx`（新建）

### 页面结构

```
┌─────────────────────────────────────────────────┐
│ 服务器: [Select 下拉选择]   [新建文件夹] [刷新]    │
├─────────────────────────────────────────────────┤
│ 路径: /root/uploads > subfolder > ...  (面包屑)  │
├─────────────────────────────────────────────────┤
│ 📁 .. (上级目录)                                  │
│ 📁 folder-a                    2024-01-15 10:30  │
│ 📄 file.txt          1.2 MB   2024-01-15 09:00  │
│ 📄 image.png         350 KB   2024-01-14 16:20  │
│ ...                                              │
├─────────────────────────────────────────────────┤
│ 共 N 项                                         │
└─────────────────────────────────────────────────┘
```

### 核心状态
- `servers: ServerConfig[]` — 服务器列表（从 configStore 获取）
- `selectedServerId: string` — 当前选中服务器
- `currentPath: string` — 当前目录路径
- `files: RemoteFileInfo[]` — 当前目录文件列表
- `loading: boolean` — 加载状态

### 核心功能
1. **选择服务器**: Select 组件，切换时自动 list 根目录（`server.remotePath`）
2. **目录导航**: 点击文件夹进入子目录，面包屑显示路径层级可点击跳转，`..` 行返回上级
3. **文件列表**: Ant Design Table，列：名称（带图标）、大小、修改时间、权限、操作
4. **新建文件夹**: Modal + Input 输入名称 → 调用 browserMkdir
5. **下载文件**: 操作列下载按钮 → 调用 browserDownload（主进程弹出 save dialog）
6. **删除**: Popconfirm 确认 → 调用 browserDelete → 刷新列表
7. **刷新**: 重新 list 当前目录

### 工具函数
- `formatBytes(bytes)` — 格式化文件大小（复用已有模式）
- 文件图标：文件夹用 `FolderOutlined`，文件根据扩展名显示不同图标
