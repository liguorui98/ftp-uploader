# FTP Uploader

一个跨平台的FTP/SFTP文件传输桌面应用，支持定时上传、文件监控、手动上传和服务器文件管理。

## 功能特性

- **多协议支持**: 支持FTP和SFTP协议
- **定时上传**: 支持间隔、每日、每周、Cron表达式四种调度模式
- **文件监控**: 自动监控指定文件夹，新文件自动上传
- **手动上传**: 支持选择文件和拖拽上传
- **服务器文件管理**: 可视化浏览远程服务器文件，支持新建文件夹、下载、删除、重命名
- **下载进度**: 实时显示下载进度、速度、剩余时间，支持取消下载
- **传输队列**: 支持并发控制和失败自动重试
- **跨平台**: 支持macOS和Windows
- **安全存储**: 服务器密码加密存储
- **系统托盘**: 支持最小化到系统托盘

## 技术栈

- **前端**: React 18 + TypeScript + Ant Design 5
- **后端**: Electron 30 + Node.js
- **构建**: Electron Forge + Vite
- **FTP客户端**: basic-ftp
- **SFTP客户端**: ssh2-sftp-client
- **文件监控**: chokidar
- **定时调度**: node-cron
- **状态管理**: Zustand
- **配置持久化**: electron-store

## 安装

### 前提条件

- Node.js 18+

## 使用说明

### 1. 配置服务器

1. 打开应用，进入"服务器"页面
2. 点击"添加服务器"
3. 填写服务器信息（主机、端口、用户名、密码等）
4. 点击"测试连接"验证配置
5. 保存服务器配置

### 2. 手动上传

1. 进入"传输"页面
2. 点击"选择文件"或拖拽文件到上传区域
3. 选择目标服务器
4. 点击"开始上传"

### 3. 定时上传

1. 进入"定时"页面
2. 点击"添加定时任务"
3. 配置任务名称、服务器、源文件路径、远程路径
4. 选择调度模式（间隔、每日、每周、Cron）
5. 保存任务

### 4. 服务器文件管理

1. 进入"文件管理"页面
2. 选择已配置的FTP/SFTP服务器
3. 浏览远程文件目录，点击文件夹进入子目录
4. 支持操作：新建文件夹、下载文件（含实时进度）、删除、重命名

### 5. 文件监控

1. 进入"监控"页面
2. 点击"添加监控"
3. 配置监控文件夹、目标服务器、远程路径
4. 可选：配置文件匹配模式
5. 保存监控配置

## 配置说明

### 调度模式

- **间隔模式**: 每隔N分钟执行一次
- **每日模式**: 每天指定时间执行
- **每周模式**: 每周指定天+时间执行
- **Cron模式**: 自定义Cron表达式

### 文件匹配模式

支持glob模式匹配，例如：
- `*.csv` - 匹配所有CSV文件
- `data_*.txt` - 匹配以data_开头的txt文件
- `**/*` - 匹配所有文件

## 常见问题

### Q: macOS提示"无法打开，因为无法验证开发者"

1. 打开"系统偏好设置" -> "安全性与隐私"
2. 点击"仍要打开"

### Q: 连接超时

1. 检查服务器地址和端口是否正确
2. 检查防火墙设置
3. 增加超时时间设置

### Q: 文件监控不工作

1. 确保监控路径存在且有读取权限
2. 检查文件匹配模式是否正确
3. 查看日志获取详细错误信息
